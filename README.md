# Home-Store-Expo_app
Home-Store is a simple and clean eCommerce mobile application developed using React Native with Expo.  

### Features
- Built with React Native (Expo)
- Organized folder structure with reusable components
- Easy to customize and extend
- Works on both Android and iOS using the Expo Go app

### How to Run
1. Clone the repository  
2. Install dependencies using `npm install`  
3. Start the application with `npx expo start`  
4. Scan the QR code with Expo Go to preview the app on your device

### Purpose
This project serves as a learning and practice app for mobile development and UI building using Expo and React Native.
